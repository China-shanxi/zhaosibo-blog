# 赵思博个人博客
16岁初二男生的简易个人小网站，基于HTML+CSS+原生JS，GitHub Pages免费部署

## 网站内容
1. 自我介绍：年龄、年级、爱好、自身现状
2. 游戏日常板块：记录游戏心得、自我反思
3. 学习随笔：记录学习心态、改正贪玩的想法
4. 放假倒计时：自定义目标日期倒计时
5. 本地留言板：访客可以留言，本地保存不丢失

## 修改方法
1. 打开index.html
2. 文章内容直接修改页面文字
3. JS顶部 targetDate 修改倒计时目标日期
4. 导航文字、页面标题均可自定义更换

## 部署教程
1. 新建GitHub仓库，上传index.html + README.md
2. 仓库设置-Settings-Pages开启静态网站
3. 分支选择main，根目录部署，生成在线访问链接
